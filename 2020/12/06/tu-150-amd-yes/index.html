mini-ITX 组装初体验暨分体水冷从入门到漏液

发表于 2020-12-06 更新于 2020-12-07

也不知道怎么就想到做这么一台东西了……只能说 YouTube 和 chiphell 害人不浅。
事前一贯地低估了难度，也没有什么工程上的备案，直接导致一翻船心态直接爆炸然后采购了多余的备件……也不知道要怎么处理，总之只总结最后装上去的东西（不然就太多了）。

……严重违反装机法（指水管接头的钱比 CPU 贵）。

一些评论：

  • CPU 实在买不到 5000 系列的了……现在定了一个 5900x，都不知道一月能不能发货。

  • 主板是从 C8I 降级下来的……C8I 手感确实好，可惜我不配，到手一天后就点不亮了，问题不明，于是退了。

  • 之后想想还是上 B550 芯片组毕竟 ITX 机箱内也没那么大空间来扩展……而 B550 mini-ITX

    • 阿苏斯：信仰、好看、贵，明确有 ECC，内存供电规格高。问题是 CPU 供电缩水还加了一个小风扇来强压温度。堆料供电上加主动散热叫发烧，这种缩水供电上加主动散热只能叫恶心人。虽然宣传上还是能做到缩最狠的水超最高的频，但总觉得日常使用个 B550 主板还要个小风扇吹供电……而且刚好这时 chiphell 有人声讨华硕纯血 ROG 人上人优先 BIOS，猛禽用户只配干等，于是还是算了。另外还有后面板 IO 选择了质量优先（供电怎么不质量优先），3A + 1C 的 USB 3 全上了 10 Gps， 剩下的数量拿 1A + 1C USB 2 来凑，而且这是唯一一款有额外温度接针的……当然还是没有额外的转速接针。

    • 雕牌：前面板 IO 缺 C 口。不过供电参数倒是不错。

    • MSI：不明确 ECC，诡异的 CPU 散热器背板阻止了你用任何奇葩散热器的想法。另外内存通道 2 设计上有些翻车。

    • 越南擎：纸面参数无限好，技嘉同规模的供电，前 IO 齐全，后 IO 1A + 1C 的 USB 3 上了 10 Gps，剩下拿 5 Gps 的USB 3 堆了 4 个数量，但看看价格（我买完又降了，现在 170 USD 了）不知道哪里还缩了……不过华擎/永擎在 ITX 也算是这几家中经验丰富的了（指怪东西多），总之还是先上了.jpg

  • 内存是镁光的 3200 ECC，参数应该是 DDR4 规范的上确界了，虽然超频性能上可能还是没有原生频率更低的条强，但在这 DDR5 准备进入市场的时间点感觉入手还是有纪念意义的x

  • GPU 是传下来的公版 2080 Ti，凭借优异的挖矿性能其价格还是在二手市场上居高不下（虽然我从没试过拿它去解非线性方程），作为初代 RTX 希望还能再战几年。

  • 硬盘 1：905P U.2 版转接 M.2 连到主板芯片组的 PCIe 3.0x4。傲腾 SSD 参数什么的懂的都懂，有一点是关于有没有 CPU 直连的说法。Intel 的技术人员在论坛上声称“虽然我没有试过但随便插不会有什么区别”，某英文论坛上的人也测试了似乎确实没什么区别。（甚至在单纯的 IO 速度方面还好了一点点，虽然不清楚具体设置。）……于是我接到了主板背部那个散热不好的 M.2 槽。

  • 硬盘 2：传下来的三星 970 evo 没有 plus。SSD 方面你永远可以相信星星星。服役快两年了寿命还剩很多，既然能用就不升级 980 pro 或是 983 了，接在准备正面散热块下的那个直连 CPU 的 PCIe 4.0x4 M.2 以 PCIe 3.0x4 兼容模式运行。

  • 硬盘 3：希捷今年新品 Exos X18 氦气盘，正好利用 TU150 那个空间充裕但是风道不是很好的 HDD 位。X18也是在希捷打算用新技术（主动加热辅助）堆到单盘 22 TB 前最后的传统技术 HDD。现在计划是和 970 evo 来个建个简单分层存储空间，合计 20 TB 还想要 4KiB 的簇大小正好利用上了 ReFS（总感觉这样简单卷的故障率会看齐 RAID 0）。不知道 ReFS 对于简单分层存储空间有没有自动旋转和加速写 HDD 部分的策略。从微软不小心露出来的资料上看（以前） ReFS 是永远只会直接写在性能层上的，超过了性能层的容量的某个百分比再转入容量层。不知道更新后又改成什么策略了。

  • 电源最初是银欣的 SX800。然后在 C8I 去世后它就和 C8I 一起去了……原因不明，到手还少线，估计是亚马逊又卖二手给我了。于是拿了海韵的 SGX-650 顶上。现在抗 3100 + 2080 Ti 双烤除了风扇起飞没什么问题，希望能抗住以后的 5900，不然又要找银欣买 1000 W 的电源了……这样我又要多出来一堆不能退的电源线了。

  • 机箱才是万恶之源……作为比某些 ATX 机箱还大的 ITX 机箱在看重极限体积 ITX 社区总是不被看好……但看到三组风扇位刚好能用量产件且不打孔地同时装下 480 规模的冷排，同时其他部件还能保有良好空间，可以看出设计师在画草图的时候绝不是照着 Ncase 放大后一顿抄的。另外较大的体积方便走背线（和顶线）使得侧透好看，且方便随时打开侧板折腾，显卡上方甚至还有放粘土人的空间（)
    ……个人认为是体积、美观度和可维护性之间的平衡。（而且顶上没通风口意味着顶上可以叠叠乐了,真正的空间利用率在于利用机箱外空间x）
